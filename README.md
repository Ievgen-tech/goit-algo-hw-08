# goit-algo-hw-08

## Домашнє завдання — Модуль 8: Дерева

### Завдання 1 — Пошук найменшого значення у BST

**Файл:** `modul1.py`

**Опис:**
Реалізація двійкового дерева пошуку (BST) з алгоритмом знаходження найменшого значення.

У BST мінімальний елемент завжди знаходиться у крайньому лівому вузлі, тому алгоритм просто рухається ліворуч від кореня до тих пір, доки не досягне вузла без лівого нащадка.

**Часова складність:** `O(h)`, де `h` — висота дерева.

---

### Запуск

```bash
python modul1.py
```

**Очікуваний вивід:**

```text
Значення у дереві: [15, 10, 20, 8, 12, 17, 25, 6]

Структура дерева:
        ┌── 25
    ┌── 20
    │   └── 17
┌── 15
│   │   ┌── 12
│   └── 10
│       └── 8
│           └── 6

Найменше значення в дереві: 6
```

---

### Додатково — Варіант для AVL-дерева

**Файл:** `modul12.py`

**Опис:**
Реалізація AVL-дерева (самобалансуючого BST) з тією ж функцією пошуку мінімального значення.

Мінімальний елемент так само шукається проходом до крайнього лівого вузла, але висота AVL-дерева підтримується збалансованою після вставок.

**Часова складність пошуку мінімуму:** `O(h)`, для AVL у середньому та найгіршому випадку `h = O(log n)`.

### Запуск AVL-варіанту

```bash
python modul12.py
```

**Очікуваний вивід:**

```text
Значення у дереві: [15, 10, 20, 8, 12, 17, 25, 6]

Структура AVL-дерева (збалансованого):
        ┌── 25
    ┌── 20
    │   └── 17
┌── 15
│   │   ┌── 12
│   └── 10
│       └── 8
│           └── 6

Найменше значення в дереві: 6
```
